## Project setup
**YOU WILL NEED TO ADD YOUR OWN API KEY, EVEN FOR THE PREVIEW TO WORK.**
Get your api key [here](https://console.developers.google.com/apis/api)
then add it to the `.env` file at project root.

```
npm install
```
